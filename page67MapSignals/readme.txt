Для минимальной комплектации создаем три файлика:
	- main.cpp;
	- hello.pro;
	- hello.bat.
	
Для того, чтобы разрабатывать в Visual Studio необходимо:
	- прописать все файлы проекта в файл *.pro;
	- прописать в файл *.bat "qmake.exe -tp vc *.pro",
	  где * это имя файла *.pro.

Выполнить файл *.bat -> На выходе появится проект Visual Studio

Примечания:
	1. Для того, чтобы проект был кроссплатворменным по всем параметрам
	   нужно и файлы держать в правильной кодировке. Для этого в notepad++
	   необходимо установить кодировку Utf-8 (без BOM). Для этого нужно,
	   чтобы в файле было еще хоть что-нибудь написано.
	2. Когда будете скидывать мне проект знайте, что исходниками являются те файлы,
	   на которые ссылается файл *.pro, сам *.pro и *.bat.
	3. Для переноса строк в *.pro файле необходимо использовать \
	   обратный слэш.
